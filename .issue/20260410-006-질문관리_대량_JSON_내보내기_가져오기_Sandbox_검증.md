# ⚙️[기능추가][Admin] 질문관리 대량 JSON 내보내기/가져오기 + Sandbox 일괄 검증

**라벨**: `작업전`
**담당자**: 

---

📝현재 문제점
---

- 질문 관리 페이지(`/admin/questions`)에서 문제를 일괄 내보내기/가져오기할 수 있는 방법이 없다
- 현재 JSON 불러오기는 문제 직접 등록 페이지(`/admin/questions/register`)에서 **단건**만 지원한다
- 대량의 문제를 등록할 때 하나씩 JSON 파일을 불러와서 등록해야 하므로 비효율적이다
- 가져오기 시 EXECUTABLE 문제의 answerSql이 실제로 실행 가능한지 사전 검증할 수 없다

🛠️해결 방안 / 제안 기능
---

- 질문 관리 목록 페이지에 **JSON 내보내기** 기능 추가
  - 필터 조건(토픽, 난이도, 실행모드)에 맞는 **전체 문제** JSON 배열 다운로드
  - 테이블 **체크박스로 선택한 문제**만 JSON 다운로드
  - `questionUuid` 포함하여 재가져오기 시 업데이트 가능
- 질문 관리 목록 페이지에 **JSON 가져오기** 기능 추가
  - JSON 배열 파일 업로드 → 서버에서 **Sandbox 일괄 검증** 자동 수행
  - EXECUTABLE 문제: DDL + 샘플데이터 적용 후 answerSql 실행 테스트
  - CONCEPT_ONLY 문제: Sandbox 검증 스킵
  - 검증 결과를 모달로 표시 (성공/실패/스킵 + 신규/업데이트 구분)
  - "전체 등록" 또는 "성공한 것만 등록" 선택 가능
- `questionUuid` 유무에 따라 **신규 등록 / 기존 업데이트** 자동 분기

⚙️작업 내용
---

- **백엔드**
  - `QuestionExportDto`, `ImportValidationResult`, `ImportItemResult`, `ImportRequest`, `ImportResult` DTO 생성
  - `QuestionImportExportService` 서비스 생성 (내보내기, 검증, 가져오기 로직)
  - `AdminQuestionController`에 export/import 엔드포인트 4개 추가
  - `ErrorCode`에 import 관련 에러코드 추가
- **프론트엔드** (Thymeleaf + JS)
  - `questions.html`에 체크박스 컬럼, 내보내기/가져오기 버튼, 검증 결과 모달 추가
  - `question-import-export.js` 신규 생성 (체크박스 관리, 내보내기, 가져오기 모달 로직)
- Entity/DB 스키마 변경 없음
