# ⚙️[기능개선][관리자] 문제 등록 가이드에 테이블 이름 작성 규칙 추가

**라벨**: `작업전`
**담당자**: 

---

📝현재 문제점
---

- 관리자가 문제를 등록할 때 `SQLD_14`, `SQLD46_01_EMP` 처럼 시험 회차·문제 번호를 테이블 이름에 그대로 사용하는 경우가 있음
- 각 문제는 독립된 Sandbox(격리된 DB)에서 실행되므로 문제 간 테이블 이름 충돌이 없는데도 불필요하게 긴 이름을 사용함
- 학습자 화면(SchemaViewer)에 `SQLD_14` 같은 시험 내부 코드가 그대로 노출되어 가독성이 떨어짐
- 단건 등록 가이드, 대량 등록 가이드, 관리자 페이지 가이드 모달 어디에도 테이블 이름 작성 지침이 없음

🛠️해결 방안 / 제안 기능
---

- 단건 가이드 MD(`question-single-guide.md`)의 `schemaDdl 작성 규칙` 섹션에 테이블 이름 작성 규칙 항목 추가
- 대량 가이드 MD(`question-bulk-guide.md`)에 동일 규칙 추가
- 관리자 단건 등록 페이지 가이드 모달(`question-register.html`)에 시각적 안내 섹션 추가
- 관리자 대량 등록 페이지 가이드 모달(`questions.html`)에 동일 섹션 추가
- AI 변환 프롬프트 텍스트에도 테이블 이름 규칙 명시하여 AI가 생성하는 JSON에서도 올바른 이름 사용 유도

⚙️작업 내용
---

- `server/PQL-Web/src/main/resources/static/docs/question-single-guide.md`: `schemaDdl 작성 규칙` 하위에 테이블 이름 작성 규칙 및 상황별 권장 이름 표 추가
- `server/PQL-Web/src/main/resources/static/docs/question-bulk-guide.md`: 동일 내용 추가 (JSON 문자열 형태 예시로 조정)
- `server/PQL-Web/src/main/resources/templates/admin/question-register.html`: 가이드 모달 UI에 `⚠️ 테이블 이름 작성 규칙` 섹션(alert + 코드 블록) 추가, AI 프롬프트 텍스트에 규칙 삽입
- `server/PQL-Web/src/main/resources/templates/admin/questions.html`: 동일하게 가이드 모달 UI + AI 프롬프트 텍스트 수정
