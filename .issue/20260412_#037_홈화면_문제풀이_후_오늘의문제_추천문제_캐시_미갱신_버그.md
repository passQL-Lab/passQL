# ❗[버그][홈화면] 문제 풀이 후 오늘의 문제·추천 문제 상태가 즉시 갱신되지 않는 버그

**라벨**: `작업전`
**담당자**: 

---

🗒️설명
---

- 오늘의 문제를 풀고 홈화면으로 돌아와도 "풀이없음" 상태가 그대로 표시됨
- 추천 문제를 풀고 홈화면으로 돌아와도 동일한 추천 목록이 유지됨
- React Query의 `staleTime`(5분) 내에 제출이 완료되어도 캐시가 갱신되지 않아 발생

🔄재현 방법
---

1. 홈화면 진입 → 오늘의 문제 또는 추천 문제 확인
2. 해당 문제를 정상적으로 풀이·제출 완료
3. 홈화면으로 복귀
4. 오늘의 문제가 여전히 "풀이없음" 상태로 표시되거나, 방금 푼 문제가 추천 목록에 그대로 남아있는 것을 확인

📸참고 자료
---

✅예상 동작
---

- 오늘의 문제를 풀고 홈화면 복귀 시 완료(초록 체크) 상태로 즉시 변경되어야 함
- 추천 문제를 풀고 홈화면 복귀 시 추천 목록이 즉시 갱신되어야 함
