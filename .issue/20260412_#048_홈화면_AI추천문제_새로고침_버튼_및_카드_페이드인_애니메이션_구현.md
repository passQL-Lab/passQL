📝현재 문제점
---

- 홈화면 AI 추천문제 섹션에 표시된 문제가 마음에 들지 않아도 다른 문제를 요청할 수 없음
- 페이지를 새로 고침하지 않으면 동일한 추천 문제 3개가 고정되어 표시됨

🛠️해결 방안 / 제안 기능
---

- "AI 추천문제" 섹션 헤더 우측에 새로고침 아이콘 버튼 추가
- 버튼 클릭 시 `GET /questions/recommendations` 재호출로 새로운 추천 문제 세트 로드
- 버튼 클릭 시 아이콘이 360도 1회전 후 멈추는 애니메이션으로 클릭 피드백 제공
- 새 문제 로드 완료 시 카드들이 위에서 순서대로 페이드인 되는 애니메이션 적용
- 로딩 중 버튼 중복 클릭 방지 처리

⚙️작업 내용
---

- `client/src/pages/Home.tsx` — `refetchRecommendations`, `isFetching` 추가, `handleRefresh` 핸들러 및 `spinning` / `refreshKey` state 구현
- `client/src/styles/components.css` — `animate-refresh-once` (1회전), `animate-card-in` (카드 순차 페이드인) 애니메이션 추가
