# 🎨[디자인][AI문제] AI문제 풀기 카테고리 카드 UI 개선

**라벨**: `작업전`
**담당자**: 

---

🖌️요청 내용
---

- AI문제 풀기 화면(`/questions`)의 토픽 카테고리 카드 UI가 단순하고 밋밋하여 모던하게 개선
- 아이콘 박스 크기 확대(44px → 52px), 반지름 강화(10px → 14px)로 시각적 존재감 강화
- 카드 전체 반지름 확대(16px → 20px 예정, 현재 16px 유지)
- hover 시 인디고 틴트 그림자(`rgba(79,70,229,0.12)`) + 테두리 색상 변경(`#818CF8`) 효과 추가
- 모바일 터치 피드백을 위한 `:active` 상태 스타일 추가 (scale + 그림자)
- 아이콘 배경색(`bg-accent-light`) Tailwind 미정의로 인해 CSS 변수(`--color-brand-light`)로 직접 처리

🎯기대 결과
---

- 카드에 아이콘 배경 박스가 명확하게 보이고, hover/touch 시 자연스러운 피드백 제공
- 웹(마우스 hover)과 모바일(터치 active) 환경 모두에서 인터랙션 동작
- 토픽 이름이 카드에 맞게 간결하게 표시됨
