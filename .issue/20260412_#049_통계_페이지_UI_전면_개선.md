📝현재 문제점
---

- 상단 요약 카드(푼 문제 / 합격 준비도 / 연속 학습)에 아이콘이 없어 각 지표의 맥락이 불명확함
- AI 영역 분석 카드의 아이콘과 제목이 세로로 쌓여 레이아웃이 어색하고 왼쪽 패딩이 과도하게 느껴짐
- AI 영역 분석 카드 아이콘이 `Lightbulb`로 AI 생성 맥락과 맞지 않음
- AI 분석 텍스트가 정적으로 표시되어 AI가 생성해주는 느낌이 없음
- `카테고리별 문제 수` 바 차트의 X축이 0~1 소수로 표기되어 정답률인지 문제 수인지 불명확하고, 제목만으로 차트의 의미 파악 불가
- `카테고리별 문제 수` 섹션명이 모호하여 사용자가 무엇을 나타내는지 이해하기 어려움
- 합격 준비도 점수 산출 기준이 사용자에게 전혀 노출되지 않아 점수 신뢰도가 낮음
- 토픽별 학습 현황 차트의 막대/색상 의미를 사용자가 직관적으로 알 수 없음

🛠️해결 방안 / 제안 기능
---

- 상단 요약 카드 각 지표 위에 lucide-react 아이콘 추가 (A안: 아이콘 위 + 숫자 + 라벨 세로 배치)
  - 푼 문제: `CheckSquare` (인디고)
  - 합격 준비도: `BarChart2` (인디고)
  - 연속 학습: `Zap` (노랑)
- AI 영역 분석 카드 레이아웃 개선: `Sparkles` 아이콘과 "AI 영역 분석" 제목을 한 줄에 인라인 배치, 본문 텍스트는 그 아래
- AI 분석 텍스트에 단어 단위 fade-in 애니메이션 적용 (blur → clear, stagger 55ms)
- 바 차트를 recharts 제거 후 daisyUI `progress` 컴포넌트 기반 커스텀 리스트로 교체
  - 섹션명: `카테고리별 문제 수` → `토픽별 학습 현황`
  - 막대 길이: 푼 문제 수 기준
  - 색상 뱃지: 정답률 % 표시 (80%↑ 초록 / 60%↑ 인디고 / 40%↑ 노랑 / 40%↓ 빨강)
  - 미학습 토픽: 회색으로 별도 표시
  - 하단 범례 추가
- 합격 준비도 라벨 오른쪽에 `?` 아이콘 버튼 추가 → 클릭 시 팝오버 표시
  - 팝오버 내용: 점수 설명 + 정답률/범위/꾸준함 세부 항목 미니 바 표시
- 토픽별 학습 현황 제목 옆 `?` 아이콘 버튼 추가 → 클릭 시 팝오버 표시
  - 팝오버 내용: 막대 길이 = 푼 문제 수, 색상 뱃지 = 정답률 설명
