---

📝현재 문제점
---

- 대량 JSON 가이드 모달의 "AI 프롬프트 복사" 버튼을 클릭해도 복사 성공 여부를 알 수 없음
  - 버튼 자체에 아무런 시각적 변화가 없어 복사됐는지 확인 불가
  - 하단 텍스트(`복사됐습니다.`)만 나타나 버튼과 피드백 위치가 불일치
- 관리자 페이지(`/admin/**`)에서 사용 중인 passQL 로고가 구버전(파란색 체크 마크 포함 디자인)으로 프론트엔드 최신 로고와 상이함

🛠️해결 방안 / 제안 기능
---

- "AI 프롬프트 복사" 버튼 클릭 시 버튼 자체를 시각적으로 변환
  - 아이콘: `copy` → `check` (체크 아이콘)
  - 텍스트: `AI 프롬프트 복사` → `복사됨`
  - 버튼 색상: 기본(`btn-ghost`) → 초록색(`btn-success`)
  - 2초 후 원래 상태로 자동 복구
- 관리자 페이지 로고를 프론트엔드 최신 로고(그라데이션 passQL 텍스트 디자인)로 교체
  - 대상 파일: `server/PQL-Web/src/main/resources/static/images/logo.png`

⚙️작업 내용
---

- 복사 버튼에 id 부여 (`copyBulkPromptBtn`, `copyBulkPromptIcon`, `copyBulkPromptLabel`)
- `copyBulkPrompt()` 함수에 `onCopied()` 내부 함수 추가 — 버튼 상태 전환 및 복구 로직
- 기존 `bulkCopySuccessMsg` 텍스트 엘리먼트 제거
- `logo.png` 최신 파일로 교체
