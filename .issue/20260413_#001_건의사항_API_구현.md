# ⚙️[기능추가][설정] 건의사항 API 구현

**라벨**: `작업전`
**담당자**: Cassiiopeia

---

📝현재 문제점
---

- 설정 화면에 건의사항 제출·조회 UI가 구현되어 있으나(#198), 백엔드 API가 미구현 상태
- 프론트는 404 응답 시 빈 목록으로 fallback 처리 중 (임시)
- 백엔드 API 없이는 실제 건의사항 제출·확인 불가

🛠️해결 방안 / 제안 기능
---

- `POST /feedback` — 건의사항 제출
- `GET /feedback/me` — 내 건의사항 목록 조회

⚙️작업 내용
---

- `FeedbackStatus` enum 추가 (PENDING / REVIEWED / APPLIED)
- `Feedback` 엔티티 추가 (`PQL-Domain-Meta`)
- `FeedbackRepository`, `FeedbackService` 구현
- `POST /feedback`, `GET /feedback/me` Controller 구현
- `ErrorCode` 추가 (`FEEDBACK_CONTENT_EMPTY`, `FEEDBACK_CONTENT_TOO_LONG`)
- Flyway 마이그레이션 `V0_0_115__add_feedback_table.sql` 추가
