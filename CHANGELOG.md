## [0.0.34] - 2026-04-09

**PR:** #50  

**New Features**
- 관리자가 AI 생성 없이 문제를 직접 등록할 수 있는 새로운 관리자 인터페이스 추가
- JSON 형식의 문제 데이터를 페이스트 또는 파일 업로드로 한 번에 입력 가능
- 관리자 화면에서 문제 등록 전 DDL과 SQL을 테스트할 수 있는 샌드박스 실행 기능 추가

**Bug Fixes**
- DDL 실행 시 멱등성 개선으로 안정성 강화

**Chores**
- 민감한 설정값이 로그에 표시되지 않도록 마스킹 처리
- API 키 관리 체계 개선
- 버전 업데이트 (v0.0.34)
