## [0.0.34] - 2026-04-09

**PR:** #50  

**New Features**
- 관리자가 AI 생성 없이 문제를 직접 등록할 수 있는 새로운 관리자 인터페이스 추가

**Chores**
- 민감한 설정값이 로그에 표시되지 않도록 마스킹 처리
- API 키 관리 체계 개선
- 버전 업데이트 (v0.0.34)
