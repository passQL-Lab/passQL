# API 변경 로그 — 홈 인사말 엔드포인트 추가

**날짜:** 2026-04-08
**이슈:** https://github.com/passQL-Lab/passQL/issues/24

## 추가된 엔드포인트

### GET /api/home/greeting

**설명:** 회원 UUID를 받아 D-day·시간대를 반영한 랜덤 인사말 한 문장을 반환한다.

**요청 파라미터:**

| 이름 | 타입 | 필수 | 설명 |
|------|------|------|------|
| memberUuid | UUID | Y | 회원 식별자 |

**응답 (200 OK):**
```json
{ "message": "테스터님, 좋은 오후예요! 오늘도 SQL 한 문제 풀어볼까요? 📚" }
```

**에러:**

| 코드 | 조건 |
|------|------|
| 404 | memberUuid에 해당하는 활성 회원이 없는 경우 |

**메시지 분기:**
- dDay == 0: 시험 당일 메시지 풀 (4종 랜덤)
- 1 ≤ dDay ≤ 7: 막판 스퍼트 메시지 풀 (5종 랜덤)
- 8 ≤ dDay ≤ 30: D-day 인식 메시지 풀 (6종 랜덤)
- dDay > 30 또는 시험 없음: 일반 인사 메시지 풀 (8종 랜덤, 시간대 반영)
