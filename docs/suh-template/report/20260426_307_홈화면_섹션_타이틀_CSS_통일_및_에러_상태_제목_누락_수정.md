# 🚀[기능개선][홈화면] 섹션 타이틀 CSS 통일 및 에러 상태 제목 누락 수정

## 개요
홈 화면 섹션 타이틀에 혼재하던 CSS 클래스를 `text-sm text-text-secondary`로 전면 통일하고, 서버 에러 시 "합격 준비도" 제목이 표시되지 않던 문제를 수정했다.

## 변경 사항

### 프론트엔드
- `client/src/pages/Home.tsx`: 섹션 타이틀 CSS 통일 및 에러 상태 제목 추가

## 주요 구현 내용

**CSS 오타 수정**
- `text-secondary` → `text-text-secondary` 수정 (학습 현황, AI 추천문제 헤더 2곳)
- `text-secondary`는 존재하지 않는 클래스로 실제 색상이 적용되지 않던 상태였음

**타이틀 통일 기준**
- 카드 내부 소제목: `<p className="text-sm text-text-secondary">`
- 독립 섹션 헤더: `<h2 className="text-sm text-text-secondary">`
- 오늘의 문제(완료/없음 상태) 타이틀: `text-text-caption` → `text-text-secondary`

**에러 상태 제목 복원**
- `progressError` 상태 카드 구조를 `grid col-span-2` → 단일 카드로 변경
- 에러 카드 상단에 "합격 준비도" 제목 추가 — 다른 정상 섹션과 레이아웃 일관성 확보

## 주의사항
- 수정 전 `text-secondary` 클래스가 일부 전역 CSS에서 정의될 경우 의도치 않게 동작할 수 있으므로, 향후 CSS 정리 시 해당 클래스 존재 여부 재확인 권장
