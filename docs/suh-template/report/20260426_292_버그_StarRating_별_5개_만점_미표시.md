# StarRating 컴포넌트 별 5개 만점 미표시

## 개요

문제 난이도를 표시하는 StarRating 컴포넌트에서 별이 5개가 아닌 3개만 렌더링되는 버그를 수정했다. `Array.from` 호출 시 length를 3으로 하드코딩한 것이 원인이었으며, 5로 수정해 최대 5개 별이 표시되도록 했다.

## 변경 사항

### 프론트엔드

- `client/src/components/StarRating.tsx`: `Array.from({ length: 3 })` → `Array.from({ length: 5 })`로 수정

## 주요 구현 내용

`StarRating` 컴포넌트는 props로 받은 `value`(1~5)를 기준으로 채워진 별·빈 별을 렌더링하는데, 배열 길이가 3으로 하드코딩되어 있어 난이도 4~5 문제에서 별이 잘려 표시됐다. length를 5로 수정해 모든 난이도 값이 정상 표시된다.

## 주의사항

- 단순 하드코딩 오류로, 비즈니스 로직 변경은 없다.
