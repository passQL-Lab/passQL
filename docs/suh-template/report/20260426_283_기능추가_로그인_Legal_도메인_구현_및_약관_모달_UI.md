# 로그인 약관 동의 UI 개선 및 Legal 도메인 구현

## 개요

로그인 화면에 이용약관·개인정보처리방침 동의 모달을 추가하고, 약관 내용을 관리하는 Legal 독립 도메인을 백엔드에 구현했다. 관리자 페이지에서 약관을 직접 편집·게시할 수 있으며, 프론트는 로그인 진입 시 약관 최신본을 조회해 모달로 표시한다.

## 변경 사항

### 백엔드 — Legal 도메인

- `server/PQL-Domain-Meta/src/main/java/com/passql/meta/entity/Legal.java`: Legal 엔티티 신규 생성 (legalType, status, content, version 필드)
- `server/PQL-Domain-Meta/.../constant/LegalType.java`: TERMS_OF_SERVICE / PRIVACY_POLICY enum
- `server/PQL-Domain-Meta/.../constant/LegalStatus.java`: DRAFT / PUBLISHED enum
- `server/PQL-Domain-Meta/.../repository/LegalRepository.java`: 타입·상태 기반 최신본 조회 레포지토리
- `server/PQL-Domain-Meta/.../service/LegalService.java`: 약관 조회·업데이트 서비스
- `server/PQL-Domain-Meta/.../dto/LegalResponse.java`: 약관 응답 DTO
- `server/PQL-Domain-Meta/.../dto/LegalUpdateRequest.java`: 약관 수정 요청 DTO (입력 검증 포함)
- `server/PQL-Web/.../controller/MetaController.java`: `GET /api/meta/legal/{type}` 퍼블릭 엔드포인트 추가
- `server/PQL-Web/.../controller/admin/AdminLegalController.java`: 관리자 약관 조회·수정 API
- `server/PQL-Web/.../controller/admin/AdminLegalControllerDocs.java`: 관리자 약관 API 문서
- `server/PQL-Web/.../config/SecurityConfig.java`: `/api/meta/legal/**` permitAll 추가
- `server/PQL-Web/.../resources/db/migration/V0_0_159__add_legal_table.sql`: legal 테이블 DDL (PostgreSQL 문법 — TIMESTAMP, UUID, ON CONFLICT 멱등성 보장)

### 백엔드 — 관리자 UI

- `server/PQL-Web/.../resources/templates/admin/legal.html`: 약관 편집·저장 관리자 화면 (CSRF 토큰, showModal 중복 호출 수정 포함)
- `server/PQL-Web/.../resources/templates/admin/layout.html`: 사이드바에 약관 관리 메뉴 추가

### 프론트엔드

- `client/src/api/legal.ts`: Legal API 클라이언트 (apiFetch 래퍼)
- `client/src/pages/Login.tsx`: 로그인 진입 시 약관 조회 → 동의 모달 표시 로직 추가

## 주요 구현 내용

Legal은 Meta 도메인 내 독립 엔티티로 설계했다. 타입(TERMS_OF_SERVICE / PRIVACY_POLICY)과 상태(DRAFT / PUBLISHED)를 분리해 관리자가 초안을 작성 후 게시할 수 있다. 퍼블릭 API(`/api/meta/legal/{type}`)는 인증 없이 PUBLISHED 최신본만 반환하며, SecurityConfig에 permitAll을 추가해 JWT 필터를 우회시킨다. Flyway 마이그레이션은 PostgreSQL 문법으로 작성되었으며 ON CONFLICT로 멱등성을 보장한다.

## 주의사항

- `V0_0_159__add_legal_table.sql`은 초기 시드 데이터(DRAFT 상태)를 포함하므로, 배포 후 관리자 화면에서 내용을 작성하고 PUBLISHED로 전환해야 실제 모달에 표시된다.
- MetaControllerDocs에 getLegal 누락이 CodeRabbit 리뷰로 발견되어 별도 커밋에서 보완됐다.
