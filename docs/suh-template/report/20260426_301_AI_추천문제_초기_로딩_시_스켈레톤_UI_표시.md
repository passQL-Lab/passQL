# 🚀[기능개선][홈] AI 추천문제 초기 로딩 시 스켈레톤 UI 표시

## 개요
홈 화면 AI 추천문제 섹션이 초기 로딩 중 아무것도 표시되지 않던 문제를 수정했다. `isLoading` 상태를 추가로 감지해 로딩 중에는 스켈레톤 카드 3개를 표시하도록 개선했다.

## 변경 사항

### 프론트엔드
- `client/src/hooks/useHome.ts`: `useRecommendations`에서 `isLoading` 노출
- `client/src/pages/Home.tsx`: 추천 문제 섹션 렌더링 조건 및 스켈레톤 추가

## 주요 구현 내용

**렌더링 조건 변경**
- 기존: `!recommendationsError && recommendations && questions.length > 0` → 로딩 중 섹션 자체가 숨겨짐
- 변경: `!recommendationsError && (recommendationsLoading || questions.length > 0)` → 로딩 중에도 섹션 표시

**스켈레톤 UI**
- `recommendationsLoading` true일 때: `h-[72px] rounded-2xl` 스켈레톤 카드 3개 표시
- `isFetching`(새로고침) 상태에서는 기존 카드 유지 — 새로고침 시 깜빡임 방지
- 로딩 완료 후에는 `key={refreshKey}` 기반 카드 목록으로 전환

## 주의사항
- 스켈레톤은 초기 로딩(`isLoading`)에만 표시되며, 새로고침(`isFetching`)에는 표시되지 않음 — 이는 의도된 동작
