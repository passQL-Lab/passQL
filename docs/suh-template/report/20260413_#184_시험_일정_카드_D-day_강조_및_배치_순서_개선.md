# ⚙️[기능개선][홈] 시험 일정 카드 D-day 강조 및 배치 순서 개선

## 개요

홈화면 시험 일정 카드가 날짜 텍스트만 표시되어 정보 전달력이 낮았던 문제를 개선했다. 인디고 배경 카드에 D-day 숫자를 전면 강조하고, 카드 배치 순서를 시험 일정 → 오늘의 문제 순으로 변경하여 시험 준비 맥락을 우선 노출했다.

## 변경 사항

### 프론트엔드

- `client/src/pages/Home.tsx`: 시험 일정 카드 UI 교체 및 카드 순서 변경

## 주요 구현 내용

**카드 UI 교체**
- 기존의 단순 텍스트 카드(`certType round회 / examDate`)를 인디고(`bg-brand`) 배경 카드로 교체
- D-day 숫자를 `text-3xl font-bold`로 전면 강조, 시험 이름은 상단에 작게, 날짜는 하단에 서브로 배치
- 우상단/우하단에 반투명 원(`bg-white/10`, `bg-white/5`) 2개를 absolute 레이어로 배치하여 단색 배경의 깊이감 추가

**D-day 계산 로직**
- 오늘과 시험일 모두 `setHours(0, 0, 0, 0)`으로 자정 기준 정규화 후 밀리초 차이를 86400000으로 나눠 일수 계산
- 양수: `D-N`, 당일: `D-Day`, 시험 후: `D+N` 형태로 표시

**카드 배치 순서 변경**
- 기존: 오늘의 문제(왼쪽) → 시험 일정(오른쪽)
- 변경: 시험 일정(왼쪽/위) → 오늘의 문제(오른쪽/아래)
- 모바일에서는 시험 일정이 오늘의 문제보다 먼저 노출됨

## 주의사항

- 시험 일정이 없는 경우(`schedule === null`) 기존 fallback 카드("선택된 일정 없음") 유지
- D-day 계산은 클라이언트 로컬 시간 기준이므로 타임존 차이가 있는 환경에서는 하루 오차 가능성 있음
