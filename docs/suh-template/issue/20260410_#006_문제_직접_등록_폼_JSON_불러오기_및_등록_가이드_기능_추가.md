# ⚙️[기능추가][Admin] 문제 직접 등록 폼 JSON 불러오기 및 등록 가이드 기능 추가

**라벨**: `작업전`
**담당자**: 

---

📝현재 문제점
---

- 문제 직접 등록 폼(`/admin/questions/register`)에 데이터를 수동으로 입력해야 해 반복 작업이 많음
- SQLD 문제 MD 파일 보유자가 문제를 등록할 때 각 필드(topicCode, difficulty, executionMode, stem, schemaDdl 등)를 하나씩 직접 채워야 하는 불편함 존재
- topicCode 목록, difficulty 기준, executionMode 판단 기준 등 입력 가이드가 없어 다른 팀원이 문제를 등록하기 어려움

🛠️해결 방안 / 제안 기능
---

- **JSON 파일 불러오기**: `.json` 파일을 업로드하면 폼 필드가 자동으로 채워지는 기능 추가
- **JSON 붙여넣기**: JSON 텍스트를 모달에 붙여넣으면 적용되는 기능 추가
- **등록 가이드 모달**: 가이드 버튼 클릭 시 topicCode 목록, JSON 스키마, AI 변환 프롬프트를 인라인으로 확인하고 복사할 수 있는 모달 제공
- **가이드 MD 파일 다운로드**: 가이드 전체 내용을 `.md` 파일로 다운로드하는 버튼 제공 (`/docs/question-register-guide.md`)
- **AI 변환 프롬프트 복사**: SQLD 문제 MD를 AI에게 주어 JSON으로 자동 변환하는 프롬프트를 클립보드로 복사하는 버튼 제공

⚙️작업 내용
---

- `question-register.html`: JSON 파일 불러오기 / 붙여넣기 버튼 및 모달 추가
- `question-register.html`: 가이드 모달 추가 (topicCode 테이블은 Thymeleaf `${topics}` 동적 렌더링)
- `question-register.html`: `applyJson()` 함수 — topicCode → UUID 매핑, executionMode 라디오 연동, 텍스트 필드 일괄 입력
- `question-register.html`: `loadJsonFile()` — 파일 업로드 → JSON 파싱 → `applyJson()` 호출
- `question-register.html`: `copyPrompt()` — Clipboard API + execCommand fallback
- `static/docs/question-register-guide.md`: JSON 명세, topicCode 목록, difficulty 기준, executionMode 판단 기준, AI 변환 프롬프트, 입출력 예시 포함 가이드 문서 신규 생성
- `topicSelect` option에 `data-code` 속성 추가 (topicCode → UUID 매핑용)
