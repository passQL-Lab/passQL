## 📝 현재 문제점

- 오답을 제출한 이후 다시 돌아가서 풀 방법이 없다.
- 틀린 문제를 반복 학습하는 것이 자격증 시험 준비의 핵심이지만, 현재 앱은 이를 전혀 지원하지 않는다.
- 마이페이지에 학습 통계는 있지만 틀린 문제 목록에서 바로 재시도로 이어지는 동선이 없다.

## 🛠️ 해결 방안 / 제안 기능

마이페이지에 틀린 문제 목록을 제공하고, 목록에서 바로 해당 문제로 이동해 다시 풀 수 있도록 한다.

- 마이페이지 > "틀린 문제" 탭: 오답 목록 최신순 표시
- 각 문제 카드에 "다시 풀기" 버튼 → 해당 문제 상세 페이지로 이동
- 다시 풀어서 정답 맞히면 목록에서 "해결됨" 표시
- 필터: 토픽별 / 최근 N일 이내 오답
