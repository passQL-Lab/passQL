📝 현재 문제점
---

- 홈 화면 진입 시 AI 추천문제 섹션이 데이터 로딩 중에는 완전히 숨겨짐
- 데이터가 도착하기 전까지 해당 영역이 비어 보여 레이아웃 점프(CLS)가 발생함
- 다른 섹션(학습 현황, 합격 준비도)은 스켈레톤 UI가 적용되어 있으나 추천문제 섹션만 누락된 상태

🛠️ 해결 방안 / 제안 기능
---

- `useRecommendations()`의 `isLoading` 상태일 때 카드 3개 크기의 스켈레톤 표시
- 새로고침(`isFetching`) 시에는 기존 카드를 유지하여 불필요한 UI 깜빡임 방지
- 에러 시에는 기존과 동일하게 섹션 전체 숨김 유지

⚙️ 작업 내용
---

- `client/src/pages/Home.tsx` — `isLoading: recommendationsLoading` 추출 및 스켈레톤 렌더링 조건 추가
- 초기 로딩 시 `skeleton h-[72px] rounded-2xl` 3개 표시 (daisyUI skeleton 클래스 활용)
