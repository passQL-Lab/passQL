📝현재 문제점
---

- passQL의 핵심 가치는 AI 기반 SQL 교육이지만, UI 전반에서 "AI" 표현이 드러나지 않음
- 홈화면 추천 문제 섹션 제목이 "추천 문제"로만 표기되어 AI 기능임을 인지하기 어려움
- 하단 네비게이션 탭이 "문제"로만 표기되어 AI 교육 서비스임을 나타내지 못함
- 문제 관련 페이지 제목에 AI 브랜딩이 없어 서비스 정체성과 불일치

🛠️해결 방안 / 제안 기능
---

- 홈화면 추천 문제 섹션 제목: `"추천 문제"` → `✦ AI 추천문제` (Sparkles 아이콘 + 텍스트)
- 하단 네비게이션 탭: `"문제"` → `"AI문제"`
- 문제 목록/상세 페이지 제목: `"문제"` → `"AI문제"`
- lucide-react의 `Sparkles` 아이콘을 AI 표시 전용 아이콘으로 사용
