# ⚙️[기능개선][문제풀이] 문제 지문 마크다운 렌더링 및 스키마 접기/펼치기 개선

**라벨**: `작업전`
**담당자**: 

---

📝현재 문제점
---

- 문제 지문(stem)에 포함된 SQL 코드 블록(` ```sql ``` `)이 마크다운으로 렌더링되지 않고 백틱 문자 그대로 노출됨
- 스키마(SchemaViewer)가 항상 펼쳐진 상태로 표시되어 문제 화면이 길어지고 가독성이 떨어짐
- 스키마를 접거나 펼치는 토글 기능 없음

🛠️해결 방안 / 제안 기능
---

- `react-markdown` 라이브러리를 활용하여 지문(stem)을 마크다운으로 렌더링
- SQL 코드 블록은 디자인 시스템에 맞는 회색 배경 코드 블록 스타일 적용
- 접힌 상태에서는 마크다운 문법 제거 후 한 줄 미리보기로 표시
- SchemaViewer에 접기/펼치기 토글 추가 (기본값: 접힘)

⚙️작업 내용
---

- `client/src/pages/QuestionDetail.tsx` — `react-markdown` 적용, 지문 렌더링 개선
- `client/src/components/SchemaViewer.tsx` — 토글 상태 추가, 기본값 접힘으로 변경
