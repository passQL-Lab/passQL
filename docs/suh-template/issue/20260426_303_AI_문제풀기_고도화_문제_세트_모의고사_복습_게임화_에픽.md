## 📝 현재 문제점

- 현재 AI 문제 풀기는 토픽별 문제를 **하나씩 개별로** 풀기만 가능하다.
- 여러 문제를 묶어서 미니 모의고사, 데일리 세트, 틀린 문제 복습 등 다양한 학습 패턴을 지원하지 못한다.
- Production 판매 수준의 학습 경험으로는 단순 개별 문제 풀기만으로는 부족하다.

## 🛠️ 해결 방안 / 제안 기능

본 에픽은 AI 문제 풀기를 단순 개별 풀기에서 **세트 기반 학습 플랫폼**으로 고도화하는 전체 범위를 정의한다.

### A. 문제 세트 / 모의고사 계열

| 세트 타입 | 설명 |
|----------|------|
| 미니 모의고사 | 토픽 골고루 섞인 5~10문제, AI가 즉석 조합 |
| 실전 모의고사 | SQLD 시험 형식 40~50문제, 시간 제한 |
| 오늘의 데일리 세트 | 매일 자동 갱신 5문제 (전 회원 동일) |
| 약점 집중 세트 | 내가 틀린 토픽 위주로 AI가 자동 구성 |
| 토픽 심화 세트 | 특정 토픽만 10문제 연속 |

### B. 복습 / 오답 계열 (마이페이지 연계)

| 기능 | 설명 |
|------|------|
| 틀린 문제 다시 풀기 | 오답 목록에서 바로 재시도 |
| 북마크 / 찜 | 문제에 별표 → 나중에 몰아보기 |
| 오답 노트 | 틀렸을 때 내 메모 남기기 |

### C. 게임화 / 동기부여 계열

| 기능 | 설명 |
|------|------|
| 연속 학습 스트릭 | 매일 1문제 이상 풀면 불꽃 표시 |
| 토픽별 숙련도 레벨 | JOIN 마스터, 서브쿼리 초보 등 레벨 표시 |
| D-day 카운트다운 | SQLD 시험일까지 남은 날 + 오늘 목표 |

### D. 문제 품질 / 콘텐츠 계열

| 기능 | 설명 |
|------|------|
| 난이도 신고 / 피드백 | "이 문제 너무 어려워요" 버튼 |
| SQLD 기출 문제 등록 | 실제 기출을 관리자가 등록 |
