# 🚀[기능개선][통계] 레이더 차트 미학습 토픽 기본값 설정으로 그물 형태 유지

- 라벨: 작업전
- 담당자: Cassiiopeia

---

📝 현재 문제점
---

- 통계 페이지의 토픽별 분석 레이더 차트에서, 학습 이력이 없는 토픽(정답률 0%)은 값이 0으로 처리되어 차트 중앙 한 점에 모임
- 학습한 토픽이 1~2개뿐일 경우 해당 꼭짓점만 외곽으로 뻗어나와 전체 차트가 선 또는 뾰족한 모양으로 표시됨
- 거미줄(그물) 형태가 아니라 사용자가 전체 토픽 커버리지를 한눈에 파악하기 어려움

🛠️ 해결 방안 / 제안 기능
---

- 미학습 토픽(정답률 0%)에 시각적 최솟값(10)을 부여하여 차트 중심에서 약간 뻗어나온 형태로 표시
- `PolarRadiusAxis`의 도메인을 `[0, 100]`으로 고정하여 recharts의 자동 축소를 방지
- 실제 학습 데이터가 없는 상태에서도 거미줄 외형이 유지되어 전체 토픽 구조를 직관적으로 파악 가능

⚙️ 작업 내용
---
