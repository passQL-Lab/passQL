# ➕[기능추가][Admin/Question] 관리자 문제 직접 등록 기능 구현

**라벨**: `작업전`
**담당자**: 

---

📝현재 문제점
---

- 관리자가 문제를 등록하려면 AI 생성 플로우(`/admin/questions/new`)를 반드시 거쳐야 한다.
- AI 생성 없이 직접 stem, schema, answerSql 등을 입력해 문제를 저장하는 경로가 없다.
- 선택지 없이 문제만 먼저 저장하고 AI가 이후에 선택지를 자동 생성하는 워크플로우가 지원되지 않는다.

🛠️해결 방안 / 제안 기능
---

- `/admin/questions/register` GET/POST 엔드포인트를 추가하여 관리자가 직접 문제 내용을 입력해 저장할 수 있도록 한다.
- `QuestionGenerateService.createQuestionOnly()`를 신규 구현하여 Question만 저장(선택지 없음)하고, AI가 추후 선택지를 자동 생성하도록 한다.
- `ExecutionMode.EXECUTABLE`일 때 schemaDdl·answerSql 필수 검증 로직을 포함한다.
- `question-register.html` 등록 폼 페이지를 구현한다(토픽, 난이도, stem, schema, answerSql, hint, choiceSetPolicy 입력).

⚙️작업 내용
---

- `AdminQuestionController.java` — `registerForm()`, `register()` 메서드 추가, `parseExecutionMode()` / `parseChoiceSetPolicy()` 헬퍼 추출
- `QuestionGenerateService.java` — `createQuestionOnly()` 신규 구현
- `question-register.html` — Thymeleaf 등록 폼 페이지 구현
- `questions.html` — 목록 페이지에 직접 등록 버튼 연결
