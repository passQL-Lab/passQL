# ➕[기능추가][AI][관리자] Qdrant 임베딩 인덱스 관리 화면 구현

**라벨**: `작업전`
**담당자**: 

---

📝현재 문제점
---

- 문제 등록 시 Qdrant 색인은 자동 수행되나 실패 여부를 관리자가 확인할 방법이 없음
- 전체 문제 재색인 기능이 Python AI 서버에 구현됐으나 관리자 화면에서 트리거할 수 없음
- DB에는 존재하지만 Qdrant에 벡터가 없는 미색인 문제를 탐지하는 수단이 없음
- 문제 목록에서 선택한 문제만 골라서 재색인하는 UI가 없음

🛠️해결 방안 / 제안 기능
---

- `/admin/embeddings` 신규 페이지 — Qdrant 컬렉션 상태 카드(총 포인트 수 / DB 문제 수 / 미색인 수) + 미색인 문제 목록 + 전체 재색인 버튼
- 미색인 문제 목록에 체크박스 + 전체 선택/해제 + 선택 색인 버튼 (기존 문제 목록 UI 패턴 재활용)
- 기존 `/admin/questions` 툴바에 "선택 색인" 버튼 추가 — 체크박스로 문제 선택 후 단건/다건 즉시 재색인
- Python AI 서버에 `POST /api/ai/index-status` 엔드포인트 추가 — Qdrant scroll로 전체 포인트 UUID 수집 후 DB UUID와 차집합 계산
- 모든 주요 경로에 디버그/INFO 로그 추가 — 색인 수집 페이지 단위, 미색인 계산 결과, 소요 시간 추적

⚙️작업 내용
---

- Python: `qdrant_client.py`에 `get_collection_info()`, `scroll_all_ids()` 추가
- Python: `ai_router.py`에 `POST /api/ai/index-status` 추가
- Python: `ai_request.py`, `ai_response.py`에 `IndexStatusRequest`, `IndexStatusResponse` 추가
- Java: `AiGatewayClient`에 `getIndexStatus()` 추가, 신규 DTO 추가
- Java: `QuestionRepository`에 `findAllQuestionUuids()` 네이티브 쿼리 추가
- Java: `AdminEmbeddingController` 신규 생성 (PQL-Web)
- Thymeleaf: `embeddings.html` 신규, `questions.html` 툴바 수정, `layout.html` 사이드바 메뉴 추가
