---

📝 현재 문제점
---

- 로그아웃 row에 value("현재 계정에서 로그아웃" 텍스트)와 action(LogOut 아이콘)이 함께 배치되어 우측에 텍스트 + 아이콘이 나란히 붙는 어색한 형태로 표시됨
- 로그아웃은 위험 액션(destructive action)임에도 불구하고 일반 네비게이션 row(건의사항, 버전 등)와 동일한 스타일로 렌더링되어 시각적 구분이 없음
- `SettingsRow` 컴포넌트의 value + action 조합 구조가 로그아웃 같은 위험 액션 표현에 적합하지 않음

🛠️ 해결 방안 / 제안 기능
---

- 로그아웃 행을 `SettingsRow` 대신 직접 `<button>`으로 렌더링하여 위험 액션 패턴 적용
- 라벨 텍스트 자체를 `text-sem-error`(빨간색)로 표시하여 위험 액션임을 직관적으로 전달
- 부가 설명 텍스트("현재 계정에서 로그아웃") 제거 — iOS, 토스 등 표준 설정 UI 패턴과 일치
- 우측 LogOut 아이콘도 동일한 빨간색으로 통일

⚙️ 작업 내용
---

- `client/src/pages/Settings.tsx` — 로그아웃 섹션 렌더링 방식 변경
  - `SettingsRow` 제거, 직접 `<button>` 렌더링으로 교체
  - 라벨 및 아이콘 색상 `text-sem-error` 적용
  - 로딩 스피너 색상도 `border-sem-error`로 통일
