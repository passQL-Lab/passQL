# 🚀[기능개선][홈] AI 추천 문제 새로고침 세션 내 중복 제외

- 라벨: 작업전
- 담당자: Cassiiopeia

---

📝 현재 문제점
---

- 홈 화면의 AI 추천 문제 새로고침 버튼을 눌러도 매번 동일한 문제가 반환된다.
- Qdrant 벡터 검색은 동일한 쿼리 벡터(오답 패턴)에서 항상 같은 Top-K 결과를 반환하기 때문이다.
- 새로고침 버튼의 UX 목적("다른 문제 보고 싶다")과 실제 동작이 불일치한다.

🛠️ 해결 방안 / 제안 기능
---

- 새로고침 버튼을 누를 때마다 이번 세션에서 이미 추천된 문제를 `excludeQuestionUuids`로 API에 전달하여 중복 제외한다.
- 브라우저 새로고침 시 자연스럽게 초기화된다(React state 초기화).
- 세션 내 중복만 제외하며, 이미 푼 문제 전체 영구 제외는 범위 밖이다(문제 고갈 방지).

⚙️ 작업 내용
---

- 백엔드: `QuestionController` — `excludeQuestionUuid`(단수) → `excludeQuestionUuids`(복수 List) 변경
- 백엔드: `RecommendationService` — `mustNotIds`에 리스트 전체 추가
- 프론트: `src/api/questions.ts` — `excludeQuestionUuids` 배열 반복 append 직렬화
- 프론트: `src/hooks/useHome.ts` — `useRecommendations(excludeUuids)` 파라미터 추가, queryKey에 포함
- 프론트: `src/pages/Home.tsx` — `seenUuids` state 추가, 응답 도착 시 누적, 새로고침 시 전달
