🖌️요청 내용
---

홈화면에서 **오늘의 문제**를 눌렀을 때 문제풀이 화면 및 결과 화면에서 다수의 UX/UI 문제가 확인되었습니다.

**1. 정답 표시 오류**
- 결과 화면에서 정답이 `B`로 표시됨 (예: 사용자가 A를 선택해 정답인 경우에도 `정답 B` 표시)
- 정답 표기 시 `A / B / C / D` 알파벳 라벨은 불필요 — 실제 정답 텍스트나 번호만 표시되면 충분

**2. 다음 문제 버튼 텍스트 부적절**
- 오늘의 문제 플로우에서 정답 확인 후 하단 버튼이 `다음 문제`로 표시됨
- 오늘의 문제는 1문제 플로우이므로 `다음 문제` 대신 `홈으로 돌아가기` 또는 `문제 목록으로` 등으로 변경 필요
- 문제를 모두 풀었을 경우 홈 화면 또는 문제 목록으로 이동해야 함

**3. 답안 제출 버튼 UI 일관성 부재**
- 문제풀이 화면의 `답안 제출하기` 버튼 스타일이 앱 전반 버튼 디자인과 일관되지 않음

**4. 답안지(결과) 보기 불편**
- 정답 확인 화면에서 답안 내용 가독성이 낮아 답안지를 확인하기 어려움

🎯기대 결과
---

- 정답 표시 시 `A/B/C/D` 알파벳 라벨 제거, 정답 내용이 명확하게 표시됨
- 오늘의 문제 완료 후 버튼이 맥락에 맞는 텍스트(홈으로 이동 등)로 표시됨
- 모든 버튼의 스타일이 앱 디자인 가이드와 일관되게 통일됨
- 정답 확인 화면의 가독성이 향상되어 답안 내용을 쉽게 확인 가능

📋참고 자료
---

- 문제풀이 화면 스크린샷: 선택지 A 선택 후 `답안 제출하기` 버튼 화면
- 정답 결과 화면 스크린샷: `정답 B` 오표시 및 `다음 문제` 버튼 확인

💡추가 요청 사항
---

- 오늘의 문제 / 상호작용 / 일반 문제 플로우별로 완료 후 이동 동작(네비게이션)을 정의하고 일관되게 처리 필요
- 결과 화면에서 선택한 답안과 정답을 함께 비교할 수 있도록 UI 개선 고려
