# ⚙️[기능개선][문제풀기] 선택지 생성 중 로딩 UI 개선 — Sparkles 아이콘 및 문구 변경

**라벨**: `작업전`
**담당자**: 

---

🗒️설명
---

- AI 선택지 생성 중 표시되는 로딩 UI가 단순 spinner와 "선택지 생성 중..." 텍스트로만 구성되어 있어, 사용자가 AI가 작업 중임을 직관적으로 인지하기 어렵다.
- AI 관련 기능임을 명확히 전달하고, 대기 경험을 개선할 필요가 있다.
- 관련 파일: `client/src/pages/QuestionDetail.tsx`, `client/src/styles/components.css`

🛠️해결 방안 / 제안 기능
---

- 로딩 아이콘을 단순 spinner에서 `Sparkles` (lucide-react) 아이콘으로 교체 — 프로젝트 AI 관련 UI 통일 기준에 맞춤
- 문구를 `"선택지 생성 중..."`에서 `"AI가 선택지를 만들고 있어요"`로 변경
- Sparkles 아이콘에 빠른 pulse 애니메이션 적용 (`animate-pulse-fast`, 1.2s) — 기본 Tailwind pulse(2s)보다 1.5배 빠른 반짝임으로 생동감 부여

✅예상 동작
---

- 선택지 생성 중 화면에 Sparkles 아이콘이 반짝이며 "AI가 선택지를 만들고 있어요" 문구가 표시된다.
- 사용자가 AI가 작업 중임을 직관적으로 인지할 수 있다.
