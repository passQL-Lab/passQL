📝현재 문제점
---

- 문제 관리 페이지(`/admin/questions`)에 체크박스 선택 후 **일괄 삭제** 기능이 없음
  - 현재 삭제는 행마다 개별 삭제 버튼으로만 가능
  - 체크박스(`question-checkbox`)와 전체선택(`selectAllCheckbox`) UI는 이미 존재하나, 선택 내보내기에만 활용 중
- 툴바 버튼이 너무 많아 한 줄에 나열되어 가독성이 저하됨
  - 현재 버튼: 가이드 / JSON 가져오기 / 필터 전체 내보내기 / 선택 내보내기 / 직접 등록 / AI 생성
  - 내보내기 관련 버튼 3개가 한 줄에 나열되어 공간 낭비

🛠️해결 방안 / 제안 기능
---

- **선택 일괄삭제 버튼 추가**
  - 체크박스로 선택된 문제들을 한 번에 삭제하는 버튼 (`선택 삭제 (N)`) 추가
  - 선택된 항목이 없으면 비활성화(disabled) 처리
  - 삭제 전 확인 다이얼로그 표시 (삭제 개수 포함)
  - 백엔드: 선택된 UUID 목록을 받아 일괄 삭제하는 API 추가 (`DELETE /admin/questions/bulk`)
  - 관련 파일
    - `server/PQL-Web/src/main/resources/templates/admin/questions.html`
    - `server/PQL-Web/src/main/resources/static/js/admin/question-import-export.js`

- **툴바 UI 개선**
  - 내보내기 계열 버튼(필터 전체 내보내기, 선택 내보내기)을 드롭다운 메뉴로 묶어 버튼 수 축소
  - 또는 내보내기 / 삭제 등 액션을 선택 항목 수에 따라 컨텍스트 액션 바로 표시하는 방식 검토
  - 주요 액션(직접 등록, AI 생성)은 툴바 우측에 강조 배치 유지
