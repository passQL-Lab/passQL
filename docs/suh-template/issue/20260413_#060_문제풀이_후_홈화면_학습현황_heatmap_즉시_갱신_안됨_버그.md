# ❗[버그][홈화면] 문제 풀이 후 학습 현황(heatmap) 즉시 갱신 안 됨

**라벨**: `작업전`
**담당자**: 

---

🗒️설명
---

- 문제를 풀고 홈 화면으로 돌아와도 학습 현황 캘린더(heatmap)와 연속 streak이 업데이트되지 않는다.
- 원인: `DailyChallenge`, `QuestionDetail`, `RecommendationPractice`에서 답안 제출 완료 시 `heatmap` / `progress` 쿼리 무효화(`invalidateQueries`)가 누락되어, `staleTime: 5분` 동안 캐시된 오래된 데이터가 그대로 표시된다.
- 관련 파일: `client/src/pages/DailyChallenge.tsx`, `client/src/pages/QuestionDetail.tsx`, `client/src/pages/RecommendationPractice.tsx`, `client/src/hooks/useHome.ts`

🔄재현 방법
---

1. 홈 화면에서 학습 현황 캘린더 확인 (오늘 날짜 풀이 없음 상태)
2. 오늘의 문제 또는 추천 문제를 풀고 제출 완료
3. 홈 화면으로 복귀
4. 학습 현황 캘린더에 오늘 풀이가 반영되지 않음 (5분 이내)

📸참고 자료
---

- 홈 화면에서 "4월 13일 풀이 없음" 툴팁이 표시됨 (다수 문제를 풀었음에도 불구하고)

✅예상 동작
---

- 문제 제출 완료 즉시 홈 화면 복귀 시 학습 현황 캘린더에 오늘 날짜 풀이 기록이 반영되어야 한다.
- 연속 streak도 즉시 갱신되어야 한다.
