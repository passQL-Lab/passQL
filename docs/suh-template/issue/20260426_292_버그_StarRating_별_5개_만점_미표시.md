# ❗[버그][난이도] StarRating 컴포넌트 별 5개 만점 미표시

🗒️ 설명
---

- `StarRating` 컴포넌트에서 별 개수가 `3`으로 하드코딩되어 있어, 난이도가 5점 만점임에도 불구하고 별이 최대 3개까지만 표시됨.
- 난이도 4~5인 문제를 표시할 때 별이 모두 채워진 채(3/3) 표시되어 정확한 난이도를 사용자에게 전달하지 못하고 있음.
- 관련 파일: `client/src/components/StarRating.tsx`

🔄 재현 방법
---

1. 문제 목록 페이지(`/questions`) 또는 홈 화면으로 이동
2. 난이도 4 또는 5인 문제 확인
3. 별 표시가 3개 전부 채워진 채 표시되는 것을 확인 (5점 만점이 아닌 3점 만점처럼 보임)

---


✅ 예상 동작
---

- 별이 5개 표시되어야 하며, `level` 값(1~5)만큼 채워진 별, 나머지는 빈 별로 표시되어야 함.
- `Array.from({ length: 5 }, ...)` 으로 수정하면 해결됨.
