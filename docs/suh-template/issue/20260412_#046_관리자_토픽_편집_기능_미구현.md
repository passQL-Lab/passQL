# ❗[버그][Admin] 관리자 페이지 토픽 편집 버튼 클릭 시 동작 없음

**라벨**: `작업전`
**담당자**: 

---

🗒️설명
---

- 관리자 페이지 토픽/태그 관리 화면에서 토픽 편집 버튼(연필 아이콘)을 클릭해도 아무 동작이 없음
- `editTopic()` 함수가 `alert(code)` 호출만 하고 실제 편집 모달/기능이 구현되지 않은 상태

🔄재현 방법
---

1. 관리자 페이지 → 토픽/태그 관리 이동
2. 토픽 목록에서 편집(연필) 버튼 클릭
3. `alert`만 뜨고 편집 UI가 나타나지 않음

📸참고 자료
---

- 관련 파일: `PQL-Web/src/main/resources/templates/admin/topics.html`
- `editTopic()` 함수 (220번째 줄): `alert('편집: ' + code)` 로만 구현되어 있음

✅예상 동작
---

- 편집 버튼 클릭 시 해당 토픽의 `display_name`, `sort_order`, `is_active` 를 수정할 수 있는 모달이 열려야 함
- 저장 시 토픽 정보가 DB에 반영되어야 함

⚙️환경 정보
---

- **OS**: macOS
- **브라우저**: Chrome
- **기기**: Web
