# ❗[버그][홈] 홈 화면 추천 문제 API 무한 반복 호출 버그

- **라벨**: 작업전
- **담당자**: Cassiiopeia

---

🗒️ 설명
---

- 홈 화면 진입 시 `/api/questions/recommendations` API가 멈추지 않고 무한으로 반복 호출됨
- `excludeQuestionUuids` 배열이 점점 늘어나며 네트워크 탭에서 수십 건의 요청이 관찰됨

🔄 재현 방법
---

1. 로그인 후 홈 화면(`/`) 진입
2. 브라우저 네트워크 탭(Fetch/XHR)에서 `/api/questions/recommendations` 필터
3. 자동으로 반복 호출되는 것을 확인

📸 참고 자료
---

- 관련 파일:
  - `client/src/hooks/useHome.ts` — `useRecommendations` 훅
  - `client/src/pages/Home.tsx` — `seenUuids` 상태 및 `handleRefresh`

✅ 예상 동작
---

- 홈 화면 진입 시 추천 문제 API는 **1회만** 호출되어야 함
- 새로고침 버튼 클릭 시에만 `excludeQuestionUuids`를 포함한 새 요청이 발생해야 함

⚙️ 환경 정보
---

- **OS**: macOS
- **브라우저**: Chrome 146
- **기기**: PC
