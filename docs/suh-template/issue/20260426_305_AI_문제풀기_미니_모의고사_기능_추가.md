## 📝 현재 문제점

- 현재 문제를 하나씩 개별로만 풀 수 있어 실전 시험 감각을 익히기 어렵다.
- SQLD 시험은 40문제를 연속으로 풀어야 하지만, 현재 앱은 그 경험을 전혀 제공하지 못한다.
- 짧은 시간에 여러 토픽을 복합적으로 점검할 수 있는 기능이 없다.

## 🛠️ 해결 방안 / 제안 기능

AI가 즉석으로 토픽을 골고루 섞어 10문제짜리 미니 모의고사를 자동 생성한다.

- "미니 모의고사 시작" 버튼 → AI가 9개 토픽에서 균등 샘플링하여 10문제 조합
- 한 문제씩 순서대로 풀기 (이전/다음 이동 없음, 실전 방식)
- 전체 완료 후 토픽별 정답률 요약 결과 화면 제공
- 풀이 중 이탈 시 진행 상태 저장 (재접속 시 이어 풀기)

## ⚙️ 작업 내용

### BE

- [ ] `QuestionSetService.generateMiniExam()` — 토픽별 균등 샘플링 10문제 조합
- [ ] `POST /api/question-sets/mini-exam` — 미니 모의고사 세트 생성
- [ ] `GET /api/question-sets/{setId}` — 세트 상태 조회 (진행 중 / 완료)
- [ ] `POST /api/question-sets/{setId}/questions/{questionUuid}/submit` — 문제별 제출
- [ ] `app_setting` 키 추가: `mini_exam.question_count` (기본값 10)

### FE

- [ ] 미니 모의고사 시작 화면 (문제 수, 예상 소요 시간 안내)
- [ ] 세트 풀기 플로우 (진행 바: "3 / 10")
- [ ] 완료 결과 화면 (토픽별 정답률 그리드, 전체 점수)
- [ ] 이탈 후 재접속 시 이어 풀기 처리
