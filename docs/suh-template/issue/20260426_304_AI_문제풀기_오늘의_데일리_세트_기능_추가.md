## 📝 현재 문제점

- 현재 사용자는 능동적으로 문제를 탐색해야만 풀 수 있다.
- 매일 앱을 열었을 때 "오늘 이것만 풀면 된다"는 명확한 목표가 없어 재방문 동기가 낮다.
- 연속 학습 습관을 만들어주는 장치가 없다.

## 🛠️ 해결 방안 / 제안 기능

매일 자동으로 갱신되는 5문제짜리 데일리 세트를 제공한다. 전 회원이 동일한 세트를 풀며, AI가 토픽 고르게 조합한다.

- 홈 화면에 "오늘의 세트" 카드 노출
- 세트 시작 → 순서대로 5문제 풀기 → 완료 시 오늘의 결과 요약 화면
- 자정 기준으로 자동 갱신 (app_setting으로 문제 수 조정 가능)
- 오늘 이미 완료한 경우 "완료" 뱃지 표시

## ⚙️ 작업 내용

### BE

- [ ] `QuestionSetService.generateDailySet()` — 날짜 시드 기반 토픽별 5문제 조합
- [ ] `GET /api/question-sets/daily` — 오늘의 세트 조회 (이미 완료 여부 포함)
- [ ] `POST /api/question-sets/{setId}/submit` — 세트 단위 제출 및 결과 저장
- [ ] `app_setting` 키 추가: `daily_set.question_count` (기본값 5)

### FE

- [ ] 홈 화면 "오늘의 세트" 카드 컴포넌트
- [ ] 세트 풀기 플로우 화면 (진행 바 + 문제 순서 표시)
- [ ] 세트 완료 결과 요약 화면 (맞힌 수 / 틀린 수 / 소요 시간)
