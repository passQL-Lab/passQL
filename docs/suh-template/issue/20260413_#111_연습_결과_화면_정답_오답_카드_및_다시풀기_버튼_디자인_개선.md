# 🎨[디자인][PracticeResult] 연습 결과 화면 정답/오답 카드 및 다시 풀기 버튼 디자인 개선

🖌️요청 내용
---

- `PracticeResult.tsx` 문제별 결과 카드의 정답/오답 시각적 구분 개선
  - 정답 카드: 옅은 초록 배경(`bg-green-50`) + 초록 테두리(`border-green-200`)
  - 오답 카드: 옅은 빨간 배경(`bg-red-50`) + 빨간 테두리(`border-red-200`)
- 정답 카드도 아코디언 열려 `다시 풀기` 버튼 접근 가능하도록 변경
- `다시 풀기` 버튼 색상 정답/오답에 따라 구분
  - 정답: 초록(`feedback-bar-next--correct`, `#22C55E`)
  - 오답: 빨간(`feedback-bar-next--error`, `#EF4444`)
- `PracticeFeedbackBar.tsx` 오답 피드백 바 하단 `다시 풀기` 버튼도 동일하게 빨간색 적용
- 결과 화면 하단 버튼 레이블 "카테고리 목록으로" → "태그 목록으로" 변경
- 결과 카드에서 신고 버튼 제거 (결과 화면에서 불필요)

🎯기대 결과
---

- 정답/오답 카드가 색상으로 즉시 구분되어 결과 파악이 빠름
- `다시 풀기` 버튼 색상이 정답/오답 컨텍스트와 일치하여 시인성 향상
- 피드백 바의 `다시 풀기` 버튼이 오답 상황임을 색상으로 명확히 전달
