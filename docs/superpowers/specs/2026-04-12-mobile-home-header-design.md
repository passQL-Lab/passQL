# 모바일 홈화면 헤더 설계

**날짜:** 2026-04-12  
**범위:** 모바일 전용 (데스크톱은 사이드바에 로고 존재, 변경 없음)

---

## 배경

모바일 홈화면에 상단 헤더가 없어 앱 정체성이 불분명하다. 데스크톱은 좌측 사이드바에 로고가 있지만 모바일은 하단 탭만 있어 상단이 비어 있다. MVP 단계에서 로고와 팀 정보를 노출하는 최소 헤더를 추가한다.

---

## 설계

### 헤더 컴포넌트 (`MobileHeader`)

| 항목 | 값 |
|------|----|
| 높이 | 56px (하단 탭과 동일) |
| 배경 | `#FFFFFF` (흰색 고정) |
| 하단 보더 | `1px solid #E5E7EB` |
| 위치 | `sticky top-0`, `z-50` |
| 노출 조건 | 모바일(`md:hidden`)에서만 표시 |

**좌측:** `logo.png` (h-6, 현재 사이드바와 동일 크기)

**우측:** 팀 이니셜 뱃지
- 텍스트: `CRR`
- 배경: `#4F46E5` (브랜드 인디고)
- 형태: 원형, 32px
- 폰트: 12px, 흰색, bold
- 클릭 시 팀 모달 오픈

---

### 팀 모달 (`TeamModal`)

클릭 시 화면 중앙에 모달 오버레이로 표시.

**모달 내용:**

```
Cold Red Rice

홍의민    Frontend
서새찬    Backend · AI
```

| 항목 | 값 |
|------|----|
| 제목 | "Cold Red Rice" (text-h2) |
| 멤버 행 | 이름(text-body) + 역할(text-secondary, 우측 정렬) |
| 닫기 | 오버레이 클릭 or 우상단 X 버튼 |
| 배경 | `card-base` 스타일 (흰색, 12px radius) |
| 오버레이 | `rgba(0,0,0,0.4)` |

---

## 레이아웃 적용

`AppLayout.tsx`의 모바일 분기에서 기존 콘텐츠 상단에 `MobileHeader` 삽입.  
데스크톱 사이드바 분기에는 변경 없음.

---

## 컴포넌트 구조

```
components/
  MobileHeader.tsx     # 헤더 + CRR 뱃지
  TeamModal.tsx        # 팀원 모달
```
