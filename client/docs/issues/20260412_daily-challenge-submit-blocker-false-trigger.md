## 🗒️ 설명

- 데일리 챌린지에서 "제출하기" 버튼을 클릭하면, 실제로 화면을 이탈하려는 것이 아님에도
  "풀이를 그만할까요?" 확인 모달이 오탐으로 표시됨
- 제출 API 응답을 받기 전까지 피드백이 없는 상태이므로, 이탈 차단 조건이 제출 중에도
  활성 상태로 유지되어 발생

## 🔄 재현 방법

1. 데일리 챌린지 문제 진입
2. 선택지를 선택한 후 "제출하기" 버튼 클릭
3. "풀이를 그만할까요?" 모달이 표시됨 (정상: 모달 없이 제출 결과 표시)

## ✅ 예상 동작

- 제출 버튼 클릭 시 모달 없이 바로 API가 호출되고 결과(피드백)가 표시되어야 함
- "풀이를 그만할까요?" 모달은 홈으로 이동하거나 실제 이탈을 시도할 때만 표시되어야 함
