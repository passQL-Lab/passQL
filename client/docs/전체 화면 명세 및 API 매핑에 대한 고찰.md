## 📝 현재 문제점

- 디자인 시안이 9장 공유되었으나 변형이 많아 어느 화면을 실제로 구현할지 합의가 필요합니다 (하단 탭 구성이 화면마다 다르고, 결과 화면이 3가지 스타일로 존재하며, 일부 화면에는 PRD 범위를 벗어난 요소가 포함되어 있습니다).
- 프론트엔드 담당자와 백엔드 담당자가 동일한 화면-API 계약을 공유해야 5일 데드라인 내 병렬 작업이 가능합니다.
- 화면별로 어떤 API가 어느 시점에 호출되는지, 어떤 데이터가 백엔드 책임이고 어떤 것이 프론트 책임인지 명문화되어 있지 않습니다.
- 디자인에 포함된 항목 중 데모에 기여하지 않거나 5일 내 구현 불가한 항목을 미리 잘라내야 작업량이 통제됩니다.

## 🛠️ 해결 방안 / 제안 기능

- 디자인 시안을 분석하여 실제 구현할 화면을 6개로 확정하고, 각 화면의 UI 요소·필요 API·상태 관리·컷오프 항목을 명시합니다.
- 프론트에서 호출할 API 15개를 우선순위(P0/P1)로 정리하여 백엔드 작업 순서의 근거로 삼습니다.
- 디자인에 등장한 항목 중 컷오프된 14개 항목과 그 사유를 명시하여 추후 재논의 시 빠르게 의사결정할 수 있도록 합니다.
- "프론트 비즈니스 로직 제로 원칙"과 "선택지 클릭 시 자동 execute 정책"을 명문화하여 향후 구현 분쟁을 차단합니다.
- 상세 명세는 `.report/20260407_프론트_화면_명세_및_API_매핑.md`에 보관하며, 본 이슈는 그 내용을 요약하여 GitHub에서 공유합니다.

### 3️⃣ 문제 상세 화면 (`/questions/:id`) — P0

**UI 요소**

- [ ] 헤더: 뒤로가기 + 토픽 뱃지 + 우측 난이도 별
- [ ] "QUESTION {N}" 라벨
- [ ] 지문 (StemCard)
- [ ] 스키마 정보 카드 (테이블 목록 + DDL/예시)
- [ ] 선택지 카드 4개 (A, B, C, D)
  - 선택지 라벨
  - 우측 상단 실행 결과 뱃지 (SUCCESS / ERROR / 미실행)
  - SQL 코드 블록
  - SUCCESS: 결과 테이블 (동적 컬럼)
  - ERROR: 에러 코드 + 메시지 박스 + [AI에게 물어보기] 버튼
- [ ] 선택된 선택지는 보더 강조 + 체크 아이콘
- [ ] 하단 sticky [제출하기] 버튼

**필요 API**

- [ ] `GET /api/questions/{id}` — 문제 + 선택지 + 스키마
- [ ] `POST /api/questions/{id}/execute` — 선택지 클릭 시 자동 호출
- [ ] `POST /api/ai/explain-error` — 실행 에러 시 AI 해설
- [ ] `POST /api/questions/{id}/submit` — 제출

**상호작용 흐름**

1. 화면 진입 → `GET /api/questions/{id}`
2. 사용자가 선택지 라디오 클릭 → 즉시 해당 선택지 execute 자동 호출
3. SUCCESS면 결과 테이블, ERROR면 에러 코드/메시지 박스 인라인 표시
4. 같은 선택지 재클릭 → 캐시된 결과 재표시 (재호출 방지)
5. 다른 선택지 클릭 → 동일하게 execute
6. ERROR 시 [AI에게 물어보기] → explain-error 호출
7. [제출하기] → submit → 결과 화면 이동

**상태 관리**

- [ ] TanStack Query: question detail (staleTime 0)
- [ ] 로컬 useState: 선택지별 실행 결과 맵 `Record<choiceKey, ExecuteResult>`
- [ ] 로컬 useState: 선택된 choiceKey

**컷오프**

- "Schema Information" 토글 (항상 펼친 상태로 표시)
- CodeMirror 6 (D5에 검토)

**컷오프**

### ⚠️ 주의사항

- [ ] **오늘의 문제 / D-day는 백엔드 책임**: `GET /api/questions/today` 응답에 `{ question, dDay }` 포함. 시험일은 `app_setting.exam.target_date`에서 백엔드가 계산. 프론트는 받기만 한다.
- [ ] 문제 카드의 "정답률 N%" 집계는 백엔드 부담이 크므로 D5에 추가 또는 컷.
- [ ] 선택지 자동 실행으로 인해 `ratelimit.execute.per_minute` 빠르게 소진 가능 → 동일 선택지 재호출 방지 클라 캐시 + 어드민에서 한도 조정 가능 (기본값 60 → 90 검토).
- [ ] 프론트에는 비즈니스 로직(날짜 계산, 정답 판정, 닉네임 생성 등)을 일체 두지 않는다.
