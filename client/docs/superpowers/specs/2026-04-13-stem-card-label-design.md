# 지문 카드 라벨 항상 표시 — 디자인 스펙

**이슈:** #224  
**날짜:** 2026-04-13  
**담당:** EM-H20

---

## 문제

`QuestionDetail` 페이지에서 지문 카드를 펼쳤을 때 `BookOpen` 아이콘이 사라져, 지문 카드와 선택지 카드가 시각적으로 구분되지 않는다.

- 접힌 상태: `BookOpen` 아이콘 표시 → 지문임을 식별 가능
- **펼친 상태: 아이콘 사라짐 → 첫 번째 선택지처럼 보임**

---

## 해결 방안

지문 카드 내부 상단에 `BookOpen` 아이콘 + `문제` 텍스트 라벨을 **항상 표시**한다.  
접힘/펼침 상태와 무관하게 고정 노출. 카드의 배경·테두리·크기 등 나머지 스타일은 변경하지 않는다.

---

## 변경 범위

**파일 1개, 변경 1곳:**

`src/pages/QuestionDetail.tsx` — 지문 카드 버튼 내부

### 현재 구조

```
<button>
  {!stemOpen && <BookOpen />}   ← 접힌 상태에서만 표시
  {stemOpen ? <마크다운 /> : <미리보기 />}
</button>
```

### 변경 후 구조

```
<button>
  <라벨 (BookOpen + "문제")>   ← 항상 표시
  {stemOpen ? <마크다운 /> : <미리보기 />}
</button>
```

---

## 라벨 스타일

디자인 시스템 `caption` 크기 기준:

| 속성 | 값 |
|------|-----|
| 아이콘 | `BookOpen` (lucide-react), 12px |
| 텍스트 | `문제` |
| 색상 | `text-brand` (`#4F46E5`) |
| 크기 | 11px (`text-xs`) |
| 굵기 | 600 (`font-semibold`) |
| 간격 | 아이콘-텍스트 `gap-1.5`, 하단 `mb-2` |

---

## 변경하지 않는 것

- 지문 카드 배경색·테두리·radius·패딩
- 선택지 카드(`ChoiceCard`) 스타일
- 접힌 상태의 미리보기 텍스트(`stemPreview`)
- 클릭 시 토글 동작
