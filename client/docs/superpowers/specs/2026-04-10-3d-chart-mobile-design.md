# 3D 차트 모바일 조작 개선

## 배경

Stats3DChart는 마우스 전용 인터랙션(드래그 회전, 호버 툴팁, 휠 줌)으로 구현되어 있어 모바일에서 조작 불가. 호버가 없으므로 바 위의 정보도 확인할 수 없음.

## 변경 사항

### 1. 기둥 위 상시 라벨

- 각 바 상단에 `카테고리명 + 정답률%` Sprite 라벨 상시 표시
- 위치: 바 상단에서 약간 위 (y = barHeight + 0.4)
- 텍스트: Canvas 텍스처, 예) "JOIN\n82%"
- 기존 바 하단 이름 라벨 제거 (중복 방지)
- 기존 HTML 호버 툴팁은 데스크톱에서 그대로 유지

### 2. 우측 하단 조이스틱 (HTML 오버레이)

- 3D Canvas 컨테이너 위에 absolute 배치, 우측 하단 (right: 16px, bottom: 16px)
- 외부 원형 패드: 64px, 반투명 배경 (`rgba(0,0,0,0.08)`), border `#E5E7EB`
- 내부 핸들: 28px, `#4F46E5` (브랜드 인디고)
- 터치/마우스 드래그로 핸들 이동 -> spherical.theta/phi 변경
- 놓으면 핸들 중앙 복귀
- 별도 React 컴포넌트 `JoystickPad.tsx`로 분리
- 콜백: `onMove(dx: number, dy: number)` -- 정규화된 -1~1 값

### 3. 터치 탭으로 바 클릭

- `touchstart` + `touchend` 이벤트 추가
- 터치 이동 거리가 10px 미만이면 탭으로 판정
- Raycaster로 탭 위치의 바 감지 -> `onCategoryClick` 실행
- 기존 마우스 클릭 로직과 동일한 UX

## 변경하지 않는 것

- 데스크톱 마우스 드래그 회전 (기존 유지)
- 데스크톱 마우스 휠 줌 (기존 유지)
- 데스크톱 호버 툴팁 (기존 유지)
- 3D 씬 구조, 바 색상/높이 로직
- 2D/3D 토글 동작
